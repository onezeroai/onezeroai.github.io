Database with SQLite

Python-SQLite

This post is about database programming with Python and SQLite. A database is a collection of data with systematic access and the relationships between these data stacks are defined. Python is work with several database systems which are listed here. SQLite is a good option among these interfaces because its simplicity and easy to use.

SQLite is a free software. All codes of this software are publicly available. Therefore, you can use every bit of SQLite codes as you wish, change, sell and use them in all your commercial / non-commercial applications.

As with all relational databases, SQLite maintains data in a table-like structure shown below:

Grades

Column-1 Column-2 Column-3
Value-11 Value-12 Value-13
Value-21 Value-22 Value-23
Value-31 Value-32 Value-33

When creating a table with SQLite, we also need to name it. For example, we named the table above as “Grades”. While working with SQLite, we will perform the operations on the data using the name of the table and the columns in this table. Therefore, understanding the structure of SQLite is of great importance.

SQLite Browser

SQLite browser is a program that allows us to view the contents of SQLite databases through a graphical interface. Thanks to this program, you can confirm whether your work on the database is working correctly and see what data is in the content of the database you have. You can download the installation file for your system from here.

Now, let’s try to create a random database:

To use the sqlite3 module, we need to import the module first. Besides, random and time are other modules we need.

import sqlite3
import random
import time

con = sqlite3.connect("randdata.db")
cursor = con.cursor()


def crt_table():
    cursor.execute("CREATE TABLE IF NOT EXISTS RNDM (KeyWord TEXT, RND_Value REAL, Time REAL)")


def add_rnd(key_list):
    val_time = time.time()
    rnd_value = random.randrange(0, 100, 1)
    rnd_keyword = random.choice(key_list)
    cursor.execute("INSERT INTO RNDM (KeyWord,RND_Value,Time) VALUES(?,?,?)",
                   (rnd_keyword, rnd_value, val_time))
    con.commit()


crt_table()
i = 0
key_list = ["java", "python", "ai", "c", "csharp"]
while (i < 20):
    add_rnd(key_list)
    time.sleep(1)
    i += 1

con.close()

Here, connect method is used to create database, it is possible to give a path for .db file. It is possible to create temporary database in RAM as follows,

  con = sqlite3.connect(':memory:')

If the randdata.db is already exists then connect method is connected to this database.

There are three important SQL commands above. The first one is CREATE TABLE. This command allows us to create a table in the database. Our second command is CREATE TABLE IF NOT EXISTS. This command allows us to create a table if it doesn’t exist. Our third command is INSERT INTO … VALUES …. This command allows us to add content to the table we create. These commands are defined as functions to call when we need it.

We use random module as follows,

  rnd_value = random.randrange(0, 100, 1)
  rnd_keyword = random.choice(key_list)

These functions give a random positive integer less than 100 and a random choice from key_list, respectively. Finally, calling defined function in a while loop create our random database.
