Web Scraping with Python

The content of a website can be easily extracted by Python with the help of requests and Beautifulsoup modules. Once you got the words or in other words data then you can analyze and manipulate it.

BeautifulSoup

First of all let’s import the modules we need,

import requests
from bs4 import BeautifulSoup

then we create a variable that holds the url of the scraping page. In our example, this page is IMDb top 250 with the url linked here. In the requests module, we use the “get” function to access the webpage, after that the content is assigned to r variable. At this point, you may use print(r.content) to verify the content is gotten. Then BeautifulSoup module with “html.parser” argument arranges the html code in r. Now soup has html code parts as well as the web content. Here, we use soup.find_all function to reach the desired sections in content.

imdburl = "https://www.imdb.com/chart/top"

r = requests.get(imdburl)
soup = BeautifulSoup(r.content, "html.parser")

import_data = soup.find_all("table", {"class":"chart full-width"})

Scraping continues with the help of the find_all until you reach the desired data. After all, the film list is printed out with the help of a for loop as follows;

film_list = (import_data[0].contents)[len(import_data[0].contents)-2]
#print(film_list)
film_list_tr = film_list.find_all("tr")
fName_list = []
for film in film_list_tr:
    fname = film.find_all("td",{"class":"titleColumn"})
    fName = fname[0].text
    fName = fName.replace("\n","")
    print(fName)
    fName_list.append(fName)

If we want to write the list in a .txt file, the code below do the job.

with open("path\imdb.txt","r+") as file:  
    for film in fName_list:
        file.write(film)
        file.write("\n")

That’s all.
